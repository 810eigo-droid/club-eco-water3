## 🚀 公開までの手順

### ステップ1：画像を作る
ChatGPT等で20枚の画像を生成し、上記の名前で書き出す。
WEBP変換は [Squoosh](https://squoosh.app) または [TinyPNG](https://tinypng.com) が便利。

### ステップ2：画像を配置
作成した20枚を `images/` フォルダに入れる。

### ステップ3：表示確認
`index.html` をブラウザで開いて確認。

### ステップ4：サーバーへアップロード
このフォルダ一式（`index.html`＋`css/`＋`js/`＋`images/`）をサーバー（Xserver、エックスドメイン、ConoHa等）にFTPでアップロード。

## 🎬 アニメーションについて

- **画像**：スクロールに合わせてふわっと表示（`.reveal` クラス）
- **サーバー画像**：浮遊アニメーション（5秒周期）
- **CTAボタン**：4秒周期でほのかなパルス
- **文字**：動かさない（読みやすさ・離脱防止のため）

## 🛠 カスタマイズのヒント

### 色を変えたい
→ `css/01_base.css` の `:root` 内の CSS変数を変更

### セクションのテキストを変えたい
→ `index.html` 内の該当セクション（`<!-- セクション XX -->` で検索）を編集

### セクションの順番を変えたい
→ `index.html` 内のセクション全体（`<section class="sec-XX">...</section>`）を切り取って移動

### 全体の最大幅を変えたい（PC表示の幅）
→ `css/01_base.css` の `--content-max` を変更（現在 480px）
